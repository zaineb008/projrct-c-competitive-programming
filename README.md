# project_C 2025-2026-README:

ce projet implémente une platforme de competitive programming ,en langage C, où l'utilisateur va choisir un problème et tester son code .

* Description générale:

interface.c                           # Programme principal de l’interface

user1.c à user7.c        # Exemples de solutions utilisateur

input1.txt à input7.txt   # Fichiers d’entrée pour chaque défi

correct1.txt à correct7.txt   # Sorties attendues

user_output.txt          # Sortie générée par l’utilisateur

Interface interactive en console avec gestion d’utilisateur

* Fonctionalité:
  
7 défis algorithmiques couvrant plusieurs domaines :

Le Compte est Bon – Calcul avec opérateurs pour atteindre une cible

Mastermind Algorithmique – Décodage de séquences numériques

Course de Robots – Recherche de chemin dans un labyrinthe

Tri Ultime – Tri conditionnel basé sur une somme

Jeu Billard – Gestion de file (FIFO)

Historique Dynamique – Gestion de liste et pile (UNDO/REDO)

Arbre Binaire de Recherche (BST) – Construction et parcours d’arbres

Système de jugement automatique comparant la sortie utilisateur avec des résultats attendus

Compilation et exécution dynamique des programmes soumis

Visualisation des énoncés et instructions pour chaque défi

Structures de données : listes, piles, files, arbres, graphes

on a aussi pensé à mettre des exemples de solution dans les fichier user.c  ( exepmle: solution pour le premier problème est dans user1.c).
